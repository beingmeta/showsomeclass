Show Some Class (SSC) is a simple web application for visualizing and copy-editing HTML content, markup, and CSS classes.

As the pointer moves, block elements are outlined; inline elements are overlined. Page break constraints are displayed using dashed borders.

Click an element to edit its tag, classes, attributes, and content.

Select a range of text to directly edit either the selected text or the underlying HTML.

Once selected, you can edit the content, merge the underlying markup elements or wrap the selection with new markup.

SSC also lets you see and change how CSS classes are being applied, highlighting all elements which match a target selector.

The target selector is displayed and changed with the text box in the upper left corner of the browser window, followed by the total number of matches and the current match.

Matching elements are highlighted; surrounding elements are faded.

The selector can be any CSS selector (e.g P.class), optionally followed by a regular expression (e.g. /pattern/) matched against element content.

Move among matches with the Tab and Shift-Tab keys or the arrow ( ) icons.

See (some of) the style rules for the target selector with the S key or the icon.

Change all of the matching elements at once with the * key or the icon.

Hide the toolbar with the Escape key or the icon.

Get it back with the Enter key or by “clicking” the top of the screen.
