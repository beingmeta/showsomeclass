Show Some Class (SSC) exposes how CSS classes and HTML elements are applied and used in a document.

The current selector is displayed (and can be changed) in the text input box on the left side of the toolbar at the top of the screen.

Matching elements are highlighted and all surrounding text is faded.

An SSC selector can be any CSS selector (e.g P.class) followed by a regular expression (e.g. /pattern/) matched against element content.

Open the toolbar with the Enter key or by clicking (tapping) at the top of the screen.

Hide the toolbar with the Escape key or the icon.

Move among matched elements with the Tab and Shift-Tab keys or the arrow ( ) icons.

See (some) style rules with the S key or the icon.

Click an element on the screen to set the current selector from its tag and classes.

Click the element again to edit its properties or content.

Select a range of text to edit it directly, including wrapping it in markup or merging whatever markup elements the range contains.

Change all the matching elements at once with the * key or the icon, specifing new classes or even a new tag.
