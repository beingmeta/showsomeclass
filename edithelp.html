Show Some Class (SSC) is a simple web application for visualizing and copy-editing HTML content, markup, and CSS classes.

Block elements are outlined; inline elements are overlined.

Click an element to edit its tag, classes, attributes, and content.

Select a range of text to directly edit either the selected text or the underlying HTML. Merge the underlying markup or wrap the selection with new markup.

SSC can also expose how CSS classes are applied to the document, highlighting elements matching the current selector.

The current selector is displayed and changed using the text box in the upper left corner.

Matching elements are highlighted; surrounding elements are faded. The selector can be any CSS selector (e.g P.class), optionally followed by a regular expression (e.g. /pattern/) matched against element content.

Move among matches with the Tab and Shift-Tab keys or the arrow ( ) icons.

See (some) style rules for the current selector with the S key or the icon.

Change the tag or class of all of the matching elements at once with the * key or the icon.

Hide the toolbar with the Escape key or the icon. Get it back with the Enter key or by “clicking” the top of the screen.
