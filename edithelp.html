This is “Show Some Class”, the sPresso copy editor for HTML (and especially sBooks) content. It's a simple web application for visualizing and copy-editing HTML content, markup, and CSS classes.

The text appears as a single scrolling page with page break constraints (important for rendering in book form) displayed using dashed borders around blocks of text.

As you move your pointer around the document, block elements are outlined; inline elements are overlined.

Click an element to edit its tag, classes, attributes, and content. Clicking also sets the current selector (see below) for highlighting and skimming similar elements.

Select a range of text with the pointer to:

  • edit the text/HTML content of your selection directly,
  • merge the underlying markup elements of your selection into a single element,
  • wrap the selection with a new markup element.

You can see and change how CSS classes are applied throughout your content by specifying a target selector. This is any CSS selector (e.g P.class) optionally followed by a regular expression (e.g. /\d+/ matches any digit string) which is matched against element content. For example, LI.paragraph/peace workers/ matches all list items with the class paragraph which contains the phrase “peace workers” in its content.

The target selector appears in the editable text box on the left of the toolbar at the top of the page.

Matching elements are highlighted and the current focus match is given a colored background and dotted red outline.

Move among matches (change the focus match) with the Tab and Shift-Tab keys or by using the arrow ( ) icons in the top toolbar.

Change the tag or class of all of the matching elements at once with the * key or the icon in the toolbar.

See (some of) the style rules for the target selector with the S key or the icon. If these rules are unique to the document (either in the body or in the title's custom.css file, you can edit them directly and your changes will be saved with your other edits.

Hide the toolbar with the Escape key or the icon. Get it back with the Enter key or by “clicking” the top of the screen.
