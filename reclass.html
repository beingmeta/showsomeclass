Specifying regular class selectors (e.g. .cite) will strip all existing classes and replace them with the specified classes.

Use plus and minus (+/-) to add or delete classes, for example .+citation.-ms339 will add the class citation and drop the class ms339 from all the matching elements, without affecting any other classes.

This can also be used to change the element tag itself, to convert (for example) some classes of divs to ps or a particular kind of span to a strong.
