Specifying a new signature will modify all elements matching the current selector (). This includes elements which specify additional classes.

When you specify a new signature with regular classnames, e.g. P.quotation.avoidbreakinside), each element will lose its current classes and be adjusted to exactly match the new signature.

Alternatively, you can use plus and minus (+/-) to add or delete particular classes, for example: .+citation.-ms339 to add the class citation and drop the class ms339 from all the matching elements, without affecting any of their other classes.

This dialog can also be used to change the element tag itself, for example, changing span.italic to just em.
